# Article Update

## User Story

1. 사용자가 디테일 페이지로 이동한다.
2. 사용자가 "수정" 버튼을 클릭한다.
3.  "기존 글 내용이 담긴" 폼이 있는 페이지를 본다.
4. 글 수정 완료 후 "제출" 버튼을 누른다.
